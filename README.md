## Installation
***
You have to get your Client id and Client secret from spotify developer section. Replace it in server.js file and SpotifyApi.js file.
```
$ git clone https://github.com/barisyasar/my-spotify-web.git
$ cd server
$ npm install
$ npm run devStart
$ cd ..
$ cd client
$ npm install
$ npm start
```
